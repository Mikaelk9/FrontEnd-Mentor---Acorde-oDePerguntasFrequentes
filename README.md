## Visão Geral

### O Desafio

Os usuários devem ser capazes de:

- Ocultar/mostrar a resposta de uma pergunta quando a pergunta for clicada
- Navegar pelas perguntas e ocultar/mostrar as respostas usando apenas o teclado
- Visualizar o layout ideal da interface dependendo do tamanho da tela do dispositivo
- Ver os estados de hover e foco para todos os elementos interativos da página

### Construído com
